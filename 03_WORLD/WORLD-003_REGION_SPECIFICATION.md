# WORLD SPECIFICATION - REGION SYSTEM

---

## 21. REGION ARCHITECTURE
Region digunakan untuk membagi District.

### 21.1 Tujuan
* Optimasi server
* Optimasi AI
* Optimasi pencarian
* Optimasi pathfinding

### 21.2 Karakteristik
* Region tidak dapat dimiliki pemain.
* Region merupakan sub-area dalam District.
* Region digunakan untuk pengelompokan administrasi dan optimasi sistem.
